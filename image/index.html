Images in HTML

Images can improve the design and the appearance of a web page.

Example

<img src="../static/img_girl.jpg" alt="Girl in a jacket">

HTML Images Syntax

In HTML, images are defined with the <img> tag.
The <img> tag is empty, it contains attributes only, and does not have a closing tag. The src attribute specifies the URL (web address) of the image
The alt attribute provides an alternate text for an image, if the user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader).
Note: The alt attribute is required. A web page will not validate correctly without it.

HTML Image Maps

The <map></map> tag defines an image-map. An image-map is an image with clickable areas. Click on the computer, the phone, or the cup of coffee in the image below:

Example

<img src="workplace.jpg" alt="Workplace" usemap="#workmap"></div> <map name="workmap"> <area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm"> <area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm"> <area shape="circle" coords="337,300,44" alt="Coffee" href="coffee.htm"> </map>

Coordinates

You must define some coordinates to be able to place the clickable area onto the image.
The coordinates come in pairs, one for the x-axis and one for the y-axis.
For example the coordinates 34, 44 is located 34 pixels from the left margin and 44 pixels from the top
