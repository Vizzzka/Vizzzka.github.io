Formating text

HTML also defines special elements for defining text with a special meaning. HTML uses elements like <b> and <i> for formatting output, like bold or italic text. Formatting elements were designed to display special types of text:
  • <b></b> - Bold text
  • <strong></strong> - Important text
  • <i></i> - Italic text
  • <em></em> - Emphasized text
  • <mark></mark> - Marked text
  • <small></small> - Small text
  • <del></del> - Deleted text
  • <ins></ins> - Inserted text
  • <sub></sub> - Subscript text
  • <sup></sup> - Superscript text

Examples

HTML <b></b>

The HTML b element defines bold text, without any extra importance. <b>This text is bold</b>

HTML <strong></strong>

The HTML strong element defines strong text, with added semantic "strong" importance. <strong>This text is strong</strong>

HTML <i></i>

The HTML i element defines italic text, without any extra importance. <b>This text is italic</b>

HTML <em></em>

The HTML em element defines emphasized text, wit any added semantic importance. <em>This text is emphasized </em>

HTML <small></small>

The HTML small element defines smaller text<small>This text is smaller</small>

HTML <mark></mark>

The HTML mark element defines marked/highlighted text<mark>This text is marked/highlighted</mark>

HTML <del></del>

The HTML b element defines deleted/removed text <del>This text is deleted/removed</del>

HTML <ins></ins>

The HTML b element defines inserted/added text <ins>This text is inserted/added</ins>
